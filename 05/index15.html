Class 5: 3d Scanning

This week we took a dive into the world of 3d printing design. This started with learning how to scan objects in the real world and make them part of the digital world. Then we learned how to 3d print models and the basics of some unique 3d printing design technologies.

3D Scanning of the Fab Lab

This is a 3d model of the fabrication lab this class was in. I made this model using a phone app called polly-cam. The app works by using the built in lidar sensor on most modern phones to make a point map of the space. It then layers on photos taken at the same time as the lidar capture.
